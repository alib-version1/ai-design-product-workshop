# Getting Started with GitHub Copilot in VS Code

GitHub Copilot is an AI coding assistant built into VS Code. You do not need any coding experience to use it for building HTML prototypes today.

## How to use it

1. Open VS Code
2. Open the GitHub Copilot Chat panel — click the chat icon in the sidebar, or press **Ctrl+Shift+I** (Windows/Linux) / **Cmd+Shift+I** (Mac)
3. Describe what you want in plain English — Copilot builds it and creates the file for you
4. The file appears in your project. Double-click it to open it in your browser.

## Tips

- Describe what you want visually: "a page with a heading and a green button that says Continue" — not technical language
- Work on one page at a time
- If the output is not right, follow up in the same chat: "make the heading bigger" or "add a back link"

## When things go wrong

| Problem | What to try |
|---------|------------|
| Nothing happening | Make sure the Copilot Chat panel is open and you have sent your message |
| The page won't open in your browser | Double-click the file Copilot created in your project panel |
| Output looks wrong | Describe what you want differently — be more specific about layout and colours |
| Links between pages do not work | Tell Copilot "the back link should go to the start page" |
| Styling looks off | Say "make this look like GOV.UK with dark text and a green button" |
| The page looks wrong or incomplete | Describe what is missing or wrong and ask Copilot to fix it |

## Try it now

1. Open the GitHub Copilot Chat panel — click the chat icon in the sidebar
2. Type: "Create a simple page with a heading that says Hello and a short paragraph. Clean and professional."
3. Double-click the file Copilot created to open it in your browser

You should see a web page. That is all there is to it.
