# Getting Started with GitHub Copilot

GitHub Copilot is an AI assistant in VS Code that turns plain English descriptions into working code. You do not need any coding experience.

## How to use it

1. Open VS Code
2. Open Copilot Chat: press Ctrl+Shift+I (Windows) or Cmd+Shift+I (Mac)
3. Describe what you want in plain English
4. Copy the code it generates into your file
5. Save the file
6. Double-click the file to open it in your browser

## Tips

- Describe what you want visually: "a page with a heading and a green button that says Continue" — not technical language
- Work on one page at a time
- Preview your work by double-clicking the .html file — it opens in your browser
- If the output is not right, tell Copilot what to change: "make the heading bigger" or "add a back link"

## When things go wrong

| Problem | What to try |
|---------|------------|
| Nothing happening | Check your file is saved with a `.html` extension |
| Output looks wrong | Describe what you want differently — be more specific about layout and colours |
| Links between pages do not work | Tell Copilot Chat "the back link should go to start.html" |
| Styling looks off | Say "make this look like GOV.UK with dark text and a green button" |
| Code looks broken | Paste the whole file into Copilot Chat and say "fix this" |

## Try it now

1. Create a file called `test.html`
2. Open Copilot Chat
3. Type: "Create a simple HTML page with a heading that says Hello and a short paragraph. Clean and professional. All CSS embedded."
4. Copy the output into your file
5. Save it
6. Double-click to open in your browser

You should see a web page. That is all there is to it.
