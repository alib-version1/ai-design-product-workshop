# Useful Links

Reference resources for design, content, accessibility, and AI in government.

## GOV.UK Design System

The design system for government services. Components, patterns, and styles.
https://design-system.service.gov.uk/

## GOV.UK Service Manual

Guidance on how to build and run good government services.
https://www.gov.uk/service-manual

## GOV.UK Content Design

Guidance on writing for GOV.UK and government services.
https://www.gov.uk/guidance/content-design

## GOV.UK Prototype Kit

A tool for quickly making interactive prototypes of government services. Useful reference even if you are not using it today.
https://prototype-kit.service.gov.uk/

## WCAG 2.2 Quick Reference

Web Content Accessibility Guidelines — the standard for accessible web content.
https://www.w3.org/WAI/WCAG22/quickref/

## GOV.UK Service Standard

The 14 points that government services are assessed against.
https://www.gov.uk/service-manual/service-standard

## Generative AI Framework for HMG

The government framework for using generative AI responsibly.
https://www.gov.uk/government/publications/generative-ai-framework-for-hmg
